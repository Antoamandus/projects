HTML (Hypertext Markup Language) is the standard markup language for creating web pages and web applications. It provides the structure and layout of the content on a webpage.

HTML Elements

HTML documents are composed of HTML elements. An element consists of a start tag, content, and an end tag (with some exceptions). Elements can be nested inside other elements to create a hierarchical structure.

For example, the following code creates a paragraph element:

    <p>This is a paragraph.</p>

HTML Tags

HTML tags are used to mark up elements in an HTML document. Tags are enclosed in angle brackets (< and >) and are usually paired with a closing tag to define the start and end of an element.

Here are a few commonly used HTML tags:

  • <h1> - <h6>: Headings
  • <p>: Paragraph
  • <a>: Anchor (for creating hyperlinks)
  • <img>: Image
  • <ul> and <li>: Unordered list and list items
  • <table> and <td>: Table and table cells
  • <div> and <span>: Generic container elements

HTML Attributes

HTML attributes provide additional information about elements. They are placed within the start tag of an element and consist of a name and a value.

For example, the following code adds a "src" attribute to an image element:

    <img src="image.jpg" alt="Description of the image">

HTML Document Structure

An HTML document has a specific structure. It starts with the <!DOCTYPE html> declaration, followed by the <html> element that contains the entire document. The document is further divided into the <head> and <body> sections.

Here's an example:

    <!DOCTYPE html>
    <html>
      <head>
        <title>Page Title</title>
      </head>
      <body>
        ------
      </body>
    </html>

Within the <head> section, you can include metadata, such as the page title, CSS stylesheets, JavaScript code, and more.

Conclusion

HTML is the foundation of the web. It allows you to structure and present content in a web browser. By learning HTML, you can create static web pages, build web applications, and gain a deeper understanding of how the web works.
