HTML TAGS AND PROPERTY

TAGS PROPERTY
<acronym> The acronym tag in HTML is used to define the acronym that gives useful information to browsers, translation systems, and search engines.
<anchor> The anchor tag in HTML is used to create a hyperlink on the webpage.
<address> Defines contact information for the author/owner of a document
<area> This area tag is used in an HTML document to map a portion of an image to make it clickable by the end-user.
<base> Specifies the base URL/target for all relative URLs in a document
<bdo> Overrides the current text direction
<bdi> The bdi tag refers to Bi-Directional Isolation. It differentiates a text from other text that may be formatted in a different direction.
<blockquote> The blockquote tag in HTML is used to display the long quotations (a section that is quoted from another source).
<canvas> It can be used to draw paths, boxes, texts, gradients, and add images.
<cite> The cite tag in HTML is used to define the title of a work. It displays the text in italic format.
<dialog> This tag is used to create a popup dialog and models on a web page. This tag is new in HTML5.
<em> Indicates emphasis on the enclosed text.
<ins> The ins tag is typically used to mark a range of text that has been added to the document.
<optgroup> This tag is used to create a group of the same category options in a drop-down list.
<input> Creates an input control element.
<param> The param tag in HTML is used to define a parameter for plug-ins which is associated with
<pre> The pre tag in HTML is used to define the block of preformatted text which preserves the text spaces.
<svg> HTML SVG Basics, & their implementation through the examples. SVG stands for Scalable Vector Graphics.
<template> The template tag in HTML is used to store the HTML code fragments, which can be cloned and inserted in an HTML document.
<xmp> The XMP tag is used to create any content in letter format.
